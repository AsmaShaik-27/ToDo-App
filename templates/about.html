📌 About MyTodo

Your smart task organizer

MyTodo is a simple and elegant todo-list web application built using Flask. It helps you stay productive by organizing tasks and managing time efficiently.

This project was created as a learning project to practice web development with Flask, Bootstrap, and Jinja2. You can add, update, and delete todos easily in a responsive interface.

Future enhancements might include login support, category tags, due dates, and priority levels!
